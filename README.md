# ENEM 2024 - Exploratory Data Analysis

Projeto de análise exploratória dos microdados do ENEM 2024 utilizando Python, Pandas, Matplotlib e Seaborn.

## Objetivos

* Explorar os dados do ENEM
* Gerar estatísticas descritivas
* Construir visualizações gráficas
* Investigar desigualdade educacional

## Tecnologias

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab
